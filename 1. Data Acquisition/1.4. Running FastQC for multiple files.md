In the directory where the Fastqc is create a file “run_many_fastqc”

A notepad file with the code

```python
@echo off
set DATA_DIR=%CD%
set OUT=%DATA_DIR%\fastqc_out
mkdir "%OUT%"
cd /d "C:\Program Files\FastQC"
for %%F in ("%DATA_DIR%\*.fastq" "%DATA_DIR%\*.fastq.gz") do (
    if exist "%%~F" (
        call "run_fastqc.bat" "%%~F" -o "%OUT%"
    )
)
echo All FastQC jobs finished.
pause
```

Save this text file with the extension .bat

Then in the terminal
```python
D:
cd \YAP_TAZ_TEAD
"C:\Program Files\FastQC\run_many_fastqc.bat"
```

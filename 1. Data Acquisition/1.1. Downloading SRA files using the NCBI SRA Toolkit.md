Easiest-Download from https://www.ebi.ac.uk/ena/browser/home

Download the SRA toolkit first from the Windows 64‑bit binary from https://github.com/ncbi/sra-tools/wiki/02.-Installing-SRA-Toolkit

1. Download the zip file from the link given above
2. Extract the zip file to C:\
3. Open a command prompt, windows key and type `cmd.exe`
4. `cd`  (change directory) to the folder where you extracted the zip file.
5. The ‘SRA tools’ folder contains a sub-folder named ‘bin’
6. Change directory to bin `cd bin`
7. Test that the toolkit is functional:

`fastq-dump --stdout -X 2 SRR390728`

1. Then proceed to download multiple files 

```python
fastq-dump --split-files SRR1810900 &&fastq-dump --split-files SRR1810907 &&fastq-dump --split-files SRR1810912
```

However, fastq-dump is slower. To bypass, you can try

```python
prefetch SRR1810900
fasterq-dump SRR1810900 --outdir C:\data\Replicate\fastq
```

You can also use threads command to increase Number of CPU threads to speed up conversion

```python
fasterq-dump SRR1810900 --threads 8 --temp C:\Replicate\temp_str
```

You can also use  **`--gzip`** to compress output and use **`--split-3`** to handle paired‑end reads into **`_1`**/**`_2`** files when appropriate

```python
fastq-dump SRR12345678 --outdir C:\data\myproject\fastq --gzip --split-3
```

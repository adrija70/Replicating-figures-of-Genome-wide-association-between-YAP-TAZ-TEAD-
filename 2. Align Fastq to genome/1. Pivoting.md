Aligning fastq of interests with reference genome needed bowtie. No alternative of bowtie. Too much hassle. so figure out how to shift to linux
So had to get WSL (Windows System Linux) and Ubuntu

Open **PowerShell as Administrator** and run:

- **`wsl --install`**

On first launch, Ubuntu will finish installation and ask you to create a Linux username and password

<img width="1496" height="955" alt="image" src="https://github.com/user-attachments/assets/6c986f84-57da-4b50-b4ec-319d80b27c43" />

In that black Ubuntu terminal, type:

- **`cd ~`**
    
    This takes you to your Linux home directory instead of **`/mnt/c/WINDOWS/system32`**
    

Run these, one after the other (you will need to enter your Ubuntu password once):

- **`sudo apt update`**
- **`sudo apt upgrade`**

Create a working folder for your project

For example:

- **`mkdir -p ~/projects/genomics`**
- **`cd ~/projects/genomics`**

Now installing bioconda

In your Ubuntu terminal:

1. Download the installer (64‑bit Linux):
    
    **`wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh`**
    
2. Run the installer:
    
    **`bash Miniconda3-latest-Linux-x86_64.sh`**
    
3. Accept the license, confirm the install location (default is fine), and answer **yes** when it asks to initialize conda.
4. Close the terminal and reopen Ubuntu so that **`conda`** is available in your PATH

Back in the Ubuntu terminal, run these in order:

`conda config --add channels defaults
conda config --add channels bioconda
conda config --add channels conda-forge`

create an environment for genomics tools and install Bowtie2:

`bashconda create -n genomics bowtie2 samtools fastqc
conda activate genomics`

Check each command:

`bashbowtie2 --version
samtools --version
fastqc --version`

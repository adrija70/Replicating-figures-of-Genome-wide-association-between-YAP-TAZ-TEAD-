# Step 3 – Aligning reads to the genome

Alignment is the process of matching each sequencing read to a position in a reference genome. After alignment, each read is associated with a chromosome or a genomic coordinate.

Once reads are aligned, we can:
- summarize signal across regions
- compare samples
- look for consistent patterns of enrichment

## Why a reference genome is needed

Alignment only makes sense if there is something to align the reads *to*. That “something” is a reference genome, which serves as a common coordinate system. It is to be kept in mind that the reference genome belongs to the same species as that of the sample. Also, it is important to keep in mind the different versions. 

## Outcome of this step

By the end of this step, the sequencing reads have been mapped to genomic coordinates.

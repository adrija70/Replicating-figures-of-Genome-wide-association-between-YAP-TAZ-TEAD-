### `devtools::install_github()`

- Installs a package **not on CRAN**
- Pulls code directly from GitHub
- Requires internet + build tools

You only run this **once per R installation**.

---

### `library(Vennerable)`

- Loads the package into your session
- Makes `Venn()` and `plot()` available

## 2. Inspecting the objects

```r
TAZ_peaks
YAP_peaks
```

**Typing an object name prints it.**

You will see:

- number of ranges
- seqnames (chr1, chr2, …)
- range widths
- metadata columns

This is **sanity checking**:

- Did the file load?
- Does it look reasonable?
- Are chromosomes consistent?
